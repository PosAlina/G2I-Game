# G2I-Game "Mechanical Heart"

## Основные ветки

* develop - самая актуальная ветка с общими изменениями
* master - стабильная ветка, отрезается в конце каждого спринта
* realease - ветка со стабильным билдом, которая должна быть отполирована к майлстоунам Лесты

### Механики:

#### Общие механики

* WASD - перемещение
* E, F - взаимодействие с объектами (например, передвижение их)
* Tab - переключение персонажей
* T - переключение между камерами (от третьего лица с управлением мышкой), фиксированные камеры
* Enter - пауза
* E - подбор предметов

#### Механики инженера

* Space - прыжок
* ЛКМ - прицел
* ПКМ - выстрел

#### Механики дочки

* Space - полёт
* Ctrl - спуститься
* Проходит через некоторые объекты

#### Механики на уровне

* Головоломка с трубами
* Есть объекты, которые видит только первый персонаж, и есть объекты, которые видит только второй персонаж
* Вращение шестерёнок при выстреле по ним влияет на мир игры
* Порталы: дочка должна активировать на E первый и второй портал, после этого персонажи могут перемещаться между ними
* Головоломка: активировать предметы в нужном порядке
* Музыкальная головоломка: дочь слышит звуки у каждой трубы, нужно инженером крутануть вентили у нужных труб в нужном порядке
* Есть зона, в которой активируются при входе и деактивируются при выходе нужные акторы
* Есть зона, которая активирует нужные акторы при входе некоторых других акторов и деактивирует при выходе
* Головоломка со слайдером: инженер подходит, начинает взаимодействие, после этого должен установить слайдер в нужную точку, чтобы лампочки горели в нужной конфигурации
