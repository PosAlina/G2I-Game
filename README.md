## Описание

Прототип уровня игры "Mechanical Heart" с начальным core-геймплеем,
на примере которого наша команда проверяет работоспособность придуманной концепции

## Механики:

* WASD - перемещение
* Space - прыжок
* Space в воздухе - паровой прыжок
* LeftCtrl - присед/выход из приседа
* E - взаимодействие с объектами (например, передвижение их)
* Tab - переключение персонажей
* T - переключение между камерами (от третьего лица с управлением мышкой), фиксированные камеры
* Второй персонаж проходит сквозь некоторые объекты
* Головоломка с трубами
